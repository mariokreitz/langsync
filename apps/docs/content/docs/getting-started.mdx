---
title: Getting Started
description: Install LangSync and scaffold your first translation workflow in under two minutes.
---

## 1. Install

```bash
pnpm add -D @langsync/cli
```

## 2. Initialize

```bash
pnpm langsync init
```

This creates a typed `langsync.config.ts` and scaffolds your locale files.

## 3. Validate

```bash
pnpm langsync validate
```
