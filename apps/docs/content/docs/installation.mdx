---
title: Installation
description: How to install LangSync in your project.
---

## Requirements

- Node.js **>= 22**
- A package manager: pnpm (recommended), npm, or yarn

## Install

```bash
pnpm add -D @langsync/cli
# or: npm install -D @langsync/cli
# or: yarn add -D @langsync/cli
```

## One-off usage with `npx`

```bash
npx @langsync/cli init
```

The installed binary is also exposed as `langsync` once the package is in
your `node_modules/.bin`:

```bash
pnpm langsync init
```
